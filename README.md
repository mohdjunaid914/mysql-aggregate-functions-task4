# SQL Task 4 – Aggregate Functions & Grouping

This project focuses on using SQL aggregate functions to summarize data from the students, courses, and enrollments tables. All queries are written and tested in MySQL Workbench.

## What This Task Covers
- Using COUNT, SUM, AVG, MIN, MAX  
- Grouping data using GROUP BY  
- Filtering grouped results using HAVING  
- Summarizing enrollments per course  
- Finding average age and monthly enrollments  
- Joining tables to create meaningful summaries  

## File Included
- `mysql-aggregate-functions-task4.sql` — contains all aggregate and grouping queries.

## Outcome
By completing this task, I learn how to summarize large tables, perform group-based calculations, and analyze data using MySQL aggregate functions.
